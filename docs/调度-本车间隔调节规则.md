# 调度-本车间隔调节规则

> 刘倩 | 2026-07-13 | 时间间隔闭环调度语义收口

---

## 1. 目标

时间间隔调度统一以“本车”为作用对象，避免一条调度记录同时解释前车、后车和本车，导致页面展示和效果确认混乱。

核心原则：

```text
每条调度记录只控制 regulatedTrainId 对应的本车。
frontTrainId 只作为间隔参考对象，不直接控制前车。
后车过近时，由后车自己的“本车-前车”记录触发调度，不在本车记录里控制后车。
```

调度页展示时，统一使用：

```text
本车放慢
本车追赶
本车扣车
本车延长停站
本车缩短停站
观察
```

不要混用“放慢后车”“追赶前车”等跨对象表达。

---

## 2. 对象定义

| 对象 | 字段 | 说明 |
|---|---|---|
| 本车 | `regulatedTrainId` / `trainId` | 当前调度命令实际作用的列车 |
| 前车 | `frontTrainId` | 本车前方参考列车，只用于计算 headway |
| 目标间隔 | `targetHeadwaySec` | 当前运行图期望发车/运行间隔 |
| 实际间隔 | `actualHeadwaySec` | 本车相对前车的实际间隔 |
| 间隔偏差 | `actualHeadwaySec - targetHeadwaySec` | 负数表示过小，正数表示过大 |

统一判断：

```text
间隔过小：本车离前车太近，本车应慢下来。
间隔过大：本车离前车太远，本车应赶上去。
本车晚点：只有在不破坏间隔的前提下追赶。
```

---

## 3. 场景与动作规则

### 3.1 本车与前车间隔过小

判断：

```text
actualHeadwaySec < targetHeadwaySec - toleranceSec
headwayDirection = TOO_SHORT
```

业务含义：

```text
本车追得太近，需要扩大本车与前车之间的间隔。
```

动作优先级：

| 本车状态 | 首选动作 | 备选动作 | 不建议动作 |
|---|---|---|---|
| 在站/低速停站 | 本车延长停站 | 本车扣车 | 本车追赶、本车缩短停站 |
| 区间运行 | 本车放慢 | 临时限速/速度偏置 | 本车追赶 |
| 信号 MA 已受限 | 观察 | 等待信号条件 | 重复下强制限速 |

后端命令建议：

| 调度意图 | 命令 | 关键 payload |
|---|---|---|
| 本车延长停站 | `EXTEND_DWELL` | `deltaDwellSec > 0`、`executeOnNextDwell=true` |
| 本车放慢 | `SPEED_BIAS` | `speedBiasRatio < 1.0` |
| 本车扣车 | `HOLD` / `HOLD_TRAIN` | `holdUntilDwellSec` |

效果确认：

```text
MA 有效限速下降 / 本车速度下降 / 下一站 headwayErrorSeconds 变小或回到容差内。
```

---

### 3.2 本车与前车间隔过大

判断：

```text
actualHeadwaySec > targetHeadwaySec + toleranceSec
headwayDirection = TOO_LONG
```

业务含义：

```text
本车落后，需要缩小本车与前车之间的间隔。
```

动作优先级：

| 本车状态 | 首选动作 | 备选动作 | 不建议动作 |
|---|---|---|---|
| 在站/低速停站 | 本车缩短停站 | 提前释放停站 | 本车延长停站、本车扣车 |
| 区间运行且 MA 允许 | 本车追赶 | 速度偏置 | 本车放慢 |
| 前方进路/MA 不允许 | 观察 | 等待或申请进路 | 强行追赶 |

后端命令建议：

| 调度意图 | 命令 | 关键 payload |
|---|---|---|
| 本车缩短停站 | `SHORTEN_DWELL` | `deltaDwellSec < 0`、`executeOnNextDwell=true` |
| 本车追赶 | `SPEED_BIAS` | `speedBiasRatio > 1.0` |
| 等待信号条件 | `HEADWAY_ADJUST` / `OBSERVE` | 保留调度意图，不强制越过信号安全约束 |

效果确认：

```text
本车停站提前释放 / 本车速度提高 / headwayErrorSeconds 变小或回到容差内。
```

---

### 3.3 本车晚点

判断：

```text
departureDelaySeconds > delayTolerance
headwayDirection = SCHEDULE_LATE
```

本车晚点不能直接等价于“追赶”。必须结合本车与前车间隔：

| 间隔状态 | 动作 |
|---|---|
| 间隔过大 | 本车追赶 / 本车缩短停站 |
| 间隔正常 | 轻微追赶或观察 |
| 间隔过小 | 不追赶，优先保持间隔 |

后端第一版处理：

```text
晚点恢复优先使用本车缩短停站；
若本车正在区间运行，可使用轻微 SPEED_BIAS 追赶；
所有追赶动作都必须服从信号 MA 和有效限速。
```

---

### 3.4 本车停站时间过长

判断：

```text
disturbanceType = DWELL_EXTENDED
dwellDeviationSec > dwellToleranceSec
```

动作：

```text
本车缩短后续停站，恢复运行图。
```

后端命令：

```text
SHORTEN_DWELL
```

效果确认：

```text
本车在后续站达到最小停站后释放，departureDelaySeconds 减小。
```

---

### 3.5 客流拥挤

拥挤策略暂作为演示与扩展，不直接改变前车/后车。

当前规则：

```text
拥挤发生在本车时，调度可降低本车目标间隔或触发本车追赶意图。
```

后续如接入客流预测，应明确是：

```text
增加后续发车密度
还是调整本车停站
还是触发线路级调度计划
```

不要在一条本车调度记录里直接控制其它车。

## 4. 策略选择矩阵

| headwayDirection | 本车状态 | 调度语义 | 推荐命令 |
|---|---|---|---|
| `TOO_SHORT` | `DWELLING` / `STOPPED` / `READY` | 本车延长停站 | `EXTEND_DWELL` |
| `TOO_SHORT` | `RUNNING` | 本车放慢 | `SPEED_BIAS`，`speedBiasRatio < 1` |
| `TOO_LONG` | `DWELLING` / `STOPPED` / `READY` | 本车缩短停站 | `SHORTEN_DWELL` |
| `TOO_LONG` | `RUNNING` | 本车追赶 | `SPEED_BIAS`，`speedBiasRatio > 1` |
| `SCHEDULE_LATE` | `DWELLING` / `STOPPED` / `READY` | 本车缩短停站 | `SHORTEN_DWELL` |
| `SCHEDULE_LATE` | `RUNNING` | 本车轻微追赶 | `SPEED_BIAS`，`speedBiasRatio > 1` |
| 无有效 headway | 任意 | 观察 | 不生成强制调度命令 |

## 5. 信号边界

本规则只调整调度侧策略输出，不改变信号/轨道职责。

调度侧可以输出：

```text
本车追赶、本车放慢、本车延长/缩短停站、本车扣车等运营意图。
```

信号侧仍然负责：

```text
进路能否建立
MA 是否可延长
有效限速是多少
前方是否占用或故障
道岔/信号机是否满足安全条件
```

因此：

```text
调度动作是否“被执行”，必须通过信号 MA、有效限速、进路状态和列车状态来确认。
调度动作不能绕过信号安全约束。
```

## 6. 前端展示口径

推荐展示结构：

```text
TR-002 间隔过小
前车：TR-001
目标/实际：300s / 180s
策略：本车放慢
动作：速度偏置 0.75 或 延长停站 10s
信号证据：MA 有效限速下降 / 进路受限
效果：剩余偏差 45s，恢复中
```

不要展示：

```text
放慢后车
追赶前车
调度前车解决本车晚点
```

统一展示：

```text
本车放慢
本车追赶
本车延长停站
本车缩短停站
本车扣车
```

## 7. 验收场景

| 场景 | 输入 | 期望命令 | 作用对象 |
|---|---|---|---|
| 本车与前车间隔过小且本车在站 | `TOO_SHORT` + `DWELLING` | `EXTEND_DWELL` | 本车 |
| 本车与前车间隔过小且本车运行中 | `TOO_SHORT` + `RUNNING` | `SPEED_BIAS`，小于 1 | 本车 |
| 本车与前车间隔过大且本车在站 | `TOO_LONG` + `DWELLING` | `SHORTEN_DWELL` | 本车 |
| 本车与前车间隔过大且本车运行中 | `TOO_LONG` + `RUNNING` | `SPEED_BIAS`，大于 1 | 本车 |
| 本车晚点且在站 | `SCHEDULE_LATE` + `DWELLING` | `SHORTEN_DWELL` | 本车 |
| 本车晚点且运行中 | `SCHEDULE_LATE` + `RUNNING` | 轻微 `SPEED_BIAS` | 本车 |
