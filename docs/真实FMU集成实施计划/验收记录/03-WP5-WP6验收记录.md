## 4. 执行命令与结果

```bash
./scripts/test-fmu-power-closed-loop.sh
```

该脚本在本次验收中的结果：

- 9300：18项Maven测试通过，0失败。
- 9200：自测通过；6项同段/异段/再生/tick测试通过，0失败。
- 9000：Linux/amd64真实FMU测试镜像内20项测试通过，0失败；其中包含新增的未知实例RESYNC重建。

独立执行`mvn -f backend/pom.xml test`时，139项中有2项既有数据基线断言失败：

1. `Phase2ApiControllerTests`仍断言供电段数量为2，而当前YAML线路数据加载5个供电段。
2. `PowerServiceTests.externalVoltageComparisonFlagsDeviatedSectionsWithoutReplacingCentralVoltage`仍断言本地计算电压大于1390 V，而当前线路电阻和负荷参数计算为1368 V。

两项失败均不经过本轮9300、9000或9200新增路径；本阶段未通过放宽业务断言掩盖它们。WP5～WP6相关的9项8080定向回归已全部通过。
